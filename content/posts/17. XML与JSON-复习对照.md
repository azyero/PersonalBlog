## 易错点清单

- DOM 会把 XML 整体加载成树，适合结构操作但占内存。
- SAX 是事件流解析，适合大文件但编程模型更复杂。
- JSON 更轻量，常用于 Web API。
- 对象映射时要注意字段名、类型、日期格式和未知字段。
