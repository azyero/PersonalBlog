# 行动力被谁损害了：自我效能感与结构化维持

> 本文基于 B 站视频《【飞鸟】我们的行动力被谁损害了？提升热情和自我效能感》整理。

## 总结

行动力不是靠骂自己、逼自己、等状态好起来获得的，而是由自我效能感、合适目标、正反馈结构和外部约束共同维持出来的。

## 视频核心观点

视频围绕一个常见问题展开：为什么很多人想做事却启动不了，启动了又三分钟热度，最后一边焦虑一边摆烂？

答案不是简单的“懒”，而是一套行动系统出了问题。

| 行动力环节 | 关键问题 | 视频给出的方向 |
| --- | --- | --- |
| 我能不能做 | 自我效能感 | 停止错误自我评价，建立可做到感 |
| 我想不想做 | 目标设置和热情 | 目标明确、可达、真实，并增加关注 |
| 我能否坚持 | 结构化维持 | 用规则、休息、环境和外部约束维持 |
| 我如何避免反噬 | 情绪驱动方式 | 少用恐惧和羞辱，多用正向反馈 |

## 1. 自我贬低会直接损害行动力

视频一开始指出，很多人行动力差之后会疯狂贬低自己。

这种自我攻击看似是在“反省”，实际可能有几种心理功能：

| 自我贬低的功能 | 为什么让人短暂舒服 |
| --- | --- |
| 攻击错误行为 | 感觉自己正在改正 |
| 用负面情绪驱动自己 | 以为焦虑和羞耻会逼自己行动 |
| 降低期待 | 以后失败时不至于太失望 |

但问题是，自我贬低本身会损害行动力。因为它会降低一个人对“我能做到”的判断。

## 2. 自我效能感：不是自信，而是“我能做这件事”

视频引入了“自我效能感”这个概念：人在特定情境下，对自己能否完成某个任务的主观判断。

它不同于泛泛的自信。

一个人可以整体上很自信，但对“登月”没有自我效能感，所以不会启动登月计划。一个赌徒可能知道中奖概率极低，但他觉得自己赌术高超，所以仍然会行动。

重点不是“我觉得结果能不能成功”，而是“我觉得自己有没有能力做这件事”。

| 概念 | 关注点 | 例子 |
| --- | --- | --- |
| 结果期待 | 这件事会不会成功 | 找工作是否有机会 |
| 效能期待 | 我有没有能力去做 | 我能不能投简历、面试、优化简历 |

很多人不是因为觉得事情绝对没机会而不行动，而是因为觉得“我做不到”。

## 3. 极端自我评价都会损害自我效能感

视频不仅反对贬低自己，也反对过度拔高自己。

如果你说自己是天才，只是过去没努力，那么一旦努力两个月没有惊艳所有人，你就会得到强烈负反馈：原来我不是天才，原来我更差。

所以正确的自我评价不是“我是废物”，也不是“我是天选之人”，而是：

> 我有一些基础，有一些短板；这件事对我有难度，但通过合适方法和持续投入，我有机会做得更好。

行动力需要一种稳定、具体、可验证的自我评价。

## 4. 目标要明确，不能只是“越高越好”

第二个重点是目标设立。

人天然被目标驱动。没有目标时，就很难获得行动力。但目标不是随便立一个就行。

视频批评了三种垃圾目标：

| 垃圾目标 | 问题 |
| --- | --- |
| 不明确的目标 | 没有牵引力，例如“成绩越高越好” |
| 太高的目标 | 长期得不到反馈，容易望而生畏 |
| 虚假的目标 | 看似努力，其实没服务真正结果 |

好的目标应该明确、分层、可反馈。

例如“我要提高成绩”不够好；“我要考某大学某专业”更明确。但如果分数差距太大，仍然需要向下拆出多个梯度目标：提升 20 分、50 分、80 分分别对应什么学校或选择。

## 5. 目标拆分的目的不是变小，而是制造可确认结果

很多人听过目标拆分，但拆错了。

视频指出，拆分的目的不是形式上把时间切小，而是让每一步都留下一个可以确认的结果，从而及时获得正反馈。

| 低效拆分 | 问题 |
| --- | --- |
| 今天写 4 小时文案，拆成 4 个 1 小时 | 有启动帮助，但结果不清晰 |
| 今天刷 100 道题 | 可能只是重复会做的题 |
| 今天学习 8 小时 | 时间长，不等于提升有效 |

| 有效拆分 | 为什么有效 |
| --- | --- |
| 写完大纲 | 有明确产出 |
| 整理出 10 个错题知识点 | 结果可检查 |
| 完成一版简历并投 3 个岗位 | 行动和结果都清晰 |
| 视频完成文案、录音、剪辑、封面各阶段 | 每个阶段都有可见成果 |

一个拆分是否有效，要看它能否让你知道：我确实推进了。

## 6. 虚假努力：用战术勤奋掩盖战略懒惰

视频用很 sharp 的方式讲了“虚假目标”：把任务做完不等于目标达成。

比如学习的目标应该是掌握知识、提高成绩，但如果一个人只是机械刷会做的题，或者写一堆对理解没帮助的笔记，他确实很忙，却没有朝真正目标推进。

判断自己是否虚假努力，可以问：

1. 我现在做的事情，是否直接服务真正目标？
2. 它有没有暴露我的薄弱点？
3. 它有没有留下可检验的结果？
4. 如果今天很忙但没有变强，问题在哪里？
5. 我是在解决困难，还是在重复容易部分来获得“我很努力”的感觉？

## 7. 热情可以被制造，但不能当长期燃料

第三个重点是热情。

视频指出，对目标的关注会提升热情。你越研究一件东西，它的价值感和可达成感越容易上升。

例如想买电脑的人，研究配置越多，越可能从“随便买个够用的”变成“我特别想要那个更好的”。学习、择校、健身、职业规划也一样：持续关注目标，会提高目标的吸引力。

但热情不能当长期燃料。

人天生喜欢新鲜，厌倦重复。现代社会很多事情都要求长期、重复、深耕。即使很喜欢一个领域，也会遇到枯燥环节。

所以行动力不能只靠“我想做”。热情是启动器，不是发动机。长期坚持需要结构。

## 8. 结构化维持：番茄钟的底层原理

视频用番茄工作法讲结构化维持，但重点不是推荐某个具体方法，而是讲原理。

番茄工作法的要点包括：

| 规则 | 作用 |
| --- | --- |
| 固定时间段只做一件事 | 降低启动难度，减少任务切换 |
| 一个番茄不可分割 | 防止自我欺骗和讨价还价 |
| 到点强制休息 | 承认注意力有限，提高可持续性 |
| 被打断就重来 | 让规则有边界，避免特例常态化 |

人很擅长给自己找借口。比如“等整点开始”“今天状态不好”“先玩一下调节心情”。这些理由看似客观，其实常常是在等待一个不存在的完美开局。

结构化规则的意义，就是减少你和自己谈判的空间。

## 9. 要学会在状态不好时开始

视频里有一个非常实用的观点：人要学会带病前行，在状态不完美的时候开始。

很多人总在等：

1. 心情好一点。
2. 睡眠好一点。
3. 精力更充沛。
4. 没有杂事打扰。
5. 刚好整点。
6. 今天更适合开始。

问题是，完美开局很少出现。如果你只能在状态完美时行动，那你实际上一年也动不了几次。

真正可持续的行动力，是允许自己以 60 分状态开始。

## 10. 休息方式也会影响行动力

视频提到一个细节：休息不是随便休息。

如果工作中途刷短视频、打游戏，很容易切换到另一套思维模式，再切回来就很困难。更好的休息方式，是让大脑放松但不进入强刺激内容，比如散步、听音乐、闭眼休息、喝水。

| 休息方式 | 对专注的影响 |
| --- | --- |
| 短视频 | 高刺激，容易停不下来 |
| 游戏 | 切换到另一套任务系统 |
| 社交聊天 | 情绪和注意力被拉走 |
| 散步听音乐 | 放松但不强占认知 |
| 闭眼休息 | 恢复注意力 |

休息是为了回来，不是为了离开。

## 11. 不要长期使用恐惧驱动

视频明确反对经常用恐惧和负面情绪驱动自己。

恐惧确实能短期让人行动。就像被追赶时，人会突然有力气。但如果你长期把学习、工作、成长和恐惧绑定在一起，最终大脑会想逃避的不是失败，而是这件事本身。

例如：

| 恐惧话术 | 长期后果 |
| --- | --- |
| 不学习以后就完了 | 一想到学习就焦虑 |
| 考不好就废了 | 考试和自我价值强绑定 |
| 不努力就是垃圾 | 行动前先被羞耻压垮 |
| 再失败就没机会了 | 更害怕启动 |

恐惧驱动很容易形成恶性循环：越焦虑越逃避，越逃避问题越严重，问题越严重越焦虑。

更好的方式是正向驱动：关注目标、提高可达成感、制造正反馈、让行动和希望绑定，而不是和恐惧绑定。

## 12. 利用他人的目光做外部约束

视频最后讲了一个很有趣的技巧：利用他人的目光。

人是社会性动物，会在他人存在时感到自己被评估。图书馆、自习室、直播学习、公开打卡，本质上都在利用这个机制。

就算实际上没人关心你在干什么，你的潜意识也会因为环境而自我约束。

甚至一个“眼睛图片”、镜子、手办，都可能带来微弱的被注视感，增强自律。

可用方法：

1. 去图书馆、自习室、咖啡馆学习。
2. 和朋友开视频自习。
3. 公开承诺一个小目标。
4. 桌面放一个象征性注视物。
5. 用打卡群或直播间制造外部约束。

重点不是假装自己有多自律，而是承认人会受环境影响，然后反过来利用环境。

## 13. 行动力系统框架

可以把这期视频总结成一个行动力系统：

| 环节 | 需要建立 | 常见破坏因素 |
| --- | --- | --- |
| 自我效能感 | 我有能力开始 | 自我贬低、过高期待、失败泛化 |
| 目标吸引力 | 我想做这件事 | 目标模糊、目标太高、目标虚假 |
| 正反馈 | 我能看到推进 | 拆分错误、只按时间计算努力 |
| 结构维持 | 我能坚持 | 规则可谈判、休息方式错误 |
| 情绪燃料 | 我愿意继续 | 恐惧驱动、羞耻驱动 |
| 外部约束 | 环境帮我自律 | 一个人在高诱惑环境里硬扛 |

行动力不是一种单点品质，而是一套系统工程。

## 14. 我的总结

这期视频最实用的地方，是它没有把行动力说成“自律的人就是强，不自律的人就是差”。它把行动力拆成了心理判断、目标设计、反馈机制、情绪燃料和环境约束。

所以真正的改变不是骂自己一句“我太懒了”，而是检查：

> 我是否相信自己能做？目标是否足够清楚？反馈是否足够及时？规则是否足够稳定？环境是否在帮我？

行动力不是靠羞辱长出来的，它更像一套被设计出来的轨道。轨道铺好，人才能比较稳地往前走。
