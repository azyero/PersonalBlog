# 迷信为何泛滥：伪相关、掌控感与心理安慰

> 本文基于 B 站视频《【飞鸟】迷信为何泛滥？玄学抽卡和转锦鲤的心理学解构》整理。

## 总结

迷信并不只是“人不够聪明”，而是人类在面对随机性、失控感、不确定性和高认知成本时，天然容易发明出来的一套低成本心理补偿机制。

## 视频核心观点

视频把迷信拆成几个来源：伪相关、失控焦虑、低成本心理安慰、宁可信其有、混淆相关与因果、拟人化解释世界。

| 迷信来源 | 核心机制 | 日常表现 |
| --- | --- | --- |
| 伪相关 | 巧合被误认为规律 | 某个动作后好运发生，于是反复做 |
| 掌控感缺失 | 无法控制结果，于是制造可控动作 | 抽卡玄学、考试前仪式 |
| 低成本安慰 | 付出很少，万一有用 | 转锦鲤、许愿、幸运物 |
| 宁可信其有 | 不确定带来焦虑，于是按存在处理 | “无意冒犯”、怕触霉头 |
| 相关误作因果 | 两件事一起出现，被当成因果 | 看到规律后过度归纳 |
| 拟人化解释 | 用“人的意志”解释复杂世界 | 神、外星人、命运安排 |

视频最后的态度很克制：迷信需要被拒绝，但一些无伤大雅、清楚知道只是心理安慰的仪式，可以作为“非欺骗性安慰剂”存在。

## 1. 迷信的第一层：伪相关

视频从著名的动物实验讲起：当奖励随机出现时，动物可能会把奖励和自己刚才的动作联系起来，于是形成某种“仪式”。

这个机制不难理解。若一个动作和好结果偶然同时出现，大脑就可能把两者连接起来。下一次没有好结果时，人还会解释为“动作做得不够标准”“姿势不对”“心不诚”，而不是怀疑二者根本无关。

这就是伪相关：两件没有因果关系的事，因为偶然相邻，被强行连接。

日常例子包括：

| 情境 | 伪相关形成 |
| --- | --- |
| 考试前穿某件衣服后考得好 | 觉得这件衣服带来好运 |
| 抽卡前做某个动作后出了稀有卡 | 觉得动作影响概率 |
| 某天没带幸运物就倒霉 | 觉得幸运物能避灾 |
| 某次转发锦鲤后事情顺利 | 觉得转发真的有效 |

伪相关不是因为人没有归纳能力，而恰恰是因为人有归纳能力。归纳能力本来用于发现规律，但面对随机事件时，它也会过度工作。

## 2. 人类、动物、AI 都可能“迷信”

视频有一个很有趣的延伸：迷信并不是人类独有的现象。只要一个系统会从经验中归纳规律，就可能在随机反馈里学到不存在的规律。

这可以推广到 AI。一个强化学习模型在训练游戏时，如果某个无关动作碰巧出现在高分局里，它也可能倾向于重复那个动作。表面看起来像“AI 发明了玄学动作”，本质仍然是错误归因。

这个角度很重要，因为它说明迷信不是道德问题，也不是智商羞辱，而是学习系统面对随机反馈时的结构性风险。

## 3. 失控感越强，越容易需要玄学

第二个原因是掌控感。

人面对无法控制的事情时，会本能地想找回一点控制感。哪怕这个控制感是假的，也能降低焦虑。

视频提到彩票实验：让一部分人自己选号，另一部分人随机拿票。虽然中奖概率一样，但自己选号的人会更不愿意低价卖掉彩票，因为他们隐约觉得“我选的”更可能中奖。

这种心理在很多领域都存在：

| 高失控领域 | 常见迷信 |
| --- | --- |
| 金融投资 | 相信风水、吉日、奇怪信号 |
| 抽卡游戏 | 垫卡、切地图、换姿势 |
| 考试升学 | 幸运笔、固定衣服、转锦鲤 |
| 创业和生意 | 开张仪式、数字偏好、求财 |
| 健康和疾病 | 祈福、偏方、玄学疗法 |

越是结果随机、反馈复杂、个人控制力有限的领域，越容易滋生玄学。

## 4. 低成本心理安慰：转一下又不亏

第三个原因是迷信行为常常看起来“成本很低”。

转发一下、许个愿、戴个幸运物、拜一拜，似乎不需要付出什么。如果结果没变，也没损失；如果结果变好，就好像赚到了。

这种心理让迷信很容易传播，因为它符合“低投入、高期待”的诱惑结构。

但视频也提醒：这种低成本入口会被利用。

很多骗局和邪教宣传并不是一开始就要求你付出巨大代价，而是先让你做一个看似没成本的小动作。等你因为巧合相信过一次，后面就开始升级要求：更虔诚、更听话、更多付出、更多金钱。

这和很多诱导式商业套路很像：先用低成本诱饵拉你入场，再逐渐提高沉没成本。

## 5. “宁可信其有”本质是焦虑管理

第四个原因是人对不确定性的厌恶。

当一件事无法被完全证伪，而它又可能带来坏结果时，人会倾向于按“它存在”来处理，以消除焦虑。

例如：

| 场景 | 行为 |
| --- | --- |
| 睡前怀疑水龙头没关 | 明知大概率关了，还是起床检查 |
| 不信灵异但怕冒犯 | 弹幕里发“无意冒犯” |
| 不确定某个忌讳真假 | 干脆避开，求安心 |

“宁可信其有”不是理性判断，而是一种情绪止痛。它处理的不是事实，而是不确定性带来的焦虑。

这也是迷信很难靠讲道理彻底消除的原因：你告诉对方概率极低，但对方要处理的是“万一呢”的情绪。

## 6. 相关不等于因果，因果还可能倒置

视频还区分了两类常见错误。

第一类是把相关当因果。两件事确实一起出现，但不代表前者导致后者。

例如某些地方人少、治安差、事故多，于是有人解释为“阴气重”。但真正因素可能是地理偏僻、管理不足、照明差、求助不便。人少只是和危险相关，不一定是危险的原因。

第二类是因果倒置。你以为 A 导致 B，实际上可能是 B 导致 A。

例如某些自然灾害多的地区宗教信仰更强，不一定说明信仰导致灾害；更可能是灾害多、失控感强，所以人们更依赖信仰。

可以用下面这个检查表识别：

1. 两件事是否只是同时出现？
2. 是否存在第三个变量同时影响二者？
3. 因果方向有没有可能反过来？
4. 这个规律是否能重复验证？
5. 失败案例是否被忽略了？

## 7. 星座和人格标签：信念会反过来塑造人

视频提到一种很重要的现象：有些东西看起来“很准”，可能是因为人先相信它，然后按照它来理解自己、塑造自己。

这不是简单的“它本来准”，而是信念在改变行为。

例如一个人相信自己属于某种星座或人格类型，他会更关注与标签一致的表现，忽略不一致的表现。久而久之，他真的会越来越像那个标签。

这可以解释很多“玄学人格系统”的黏性：

| 机制 | 作用 |
| --- | --- |
| 选择性注意 | 只记得说中的部分 |
| 自我解释 | 把行为往标签上靠 |
| 社交反馈 | 别人也按标签看你 |
| 行为塑造 | 逐渐活成标签描述 |

所以“准”不一定来自预测能力，也可能来自自我实现。

## 8. 拟人化解释：用人模型理解复杂世界

最后一个原因是：人喜欢把复杂事物拟人化。

人脑最熟悉的模型是“人”。当一个系统太复杂时，把它想象成一个有意志、有情绪、有动机的人，会极大降低理解成本。

于是自然灾害可以被解释成神发怒，公司决策可以被解释成“这家公司坏”，国家行为可以被解释成“这个国家性格如何”，不明飞行现象可以被解释成外星人安排。

拟人化的好处是理解快，坏处是容易过度简化。

| 复杂对象 | 拟人化解释 | 更准确的分析方向 |
| --- | --- | --- |
| 公司 | 它是好人或坏人 | 盈利模式、组织结构、管理层博弈 |
| 国家 | 它有什么性格 | 制度、利益、历史路径、战略环境 |
| 自然灾害 | 神或命运的惩罚 | 气候、地质、生态和概率 |
| 市场波动 | 某种神秘意志 | 信息、资金、预期、政策和随机性 |

迷信常常不是完全没有逻辑，而是使用了一套过于省力的解释模型。

## 9. 对迷信的态度：拒绝迷信，但允许清醒的仪式感

视频最后给出的态度很成熟：原则上要拒绝迷信，但并非所有仪式感都必须赶尽杀绝。

如果一个人清楚知道某个行为只是心理安慰，而且它不替代理性行动、不带来实际损害，那么它可以像“非欺骗性安慰剂”一样发挥作用。

例如：

| 可以接受 | 需要警惕 |
| --- | --- |
| 考前带幸运笔，让自己安心 | 以为不复习也能靠幸运笔过 |
| 对流星许愿，当作情绪表达 | 把人生希望寄托给许愿 |
| 戴幸运物减少焦虑 | 因为没戴幸运物就拒绝出门 |
| 转发祝福当作娱乐 | 被人诱导付费、入教、服从 |

关键不是有没有仪式，而是你是否知道它只是仪式。

## 10. 可迁移框架：识别迷信的六个问题

遇到某种玄学说法，可以问：

1. 这个结论来自多少样本？
2. 有没有记录失败案例？
3. 这只是相关，还是有因果机制？
4. 有没有可能是因果倒置？
5. 我相信它，是因为证据，还是因为我需要安心？
6. 如果它开始要求我付出更多金钱、服从或关系代价，我是否还能退出？

前四个问题处理事实，后两个问题处理心理和风险。

## 11. 拓展：为什么现代社会仍然迷信泛滥

科学教育提高了，但迷信没有消失，因为现代社会也制造了更多失控感。

| 现代处境 | 对应迷信土壤 |
| --- | --- |
| 工作竞争强 | 求运势、求贵人、求转机 |
| 投资市场复杂 | 玄学指标、风水、神秘信号 |
| 亲密关系不稳定 | 星座合盘、塔罗占卜 |
| 身体焦虑增加 | 偏方、能量疗法 |
| 信息过载 | 阴谋论和神秘解释 |
| 算法推荐 | 越看越相信，形成回音室 |

迷信不是古代残留物，而是失控社会里的情绪产品。

## 12. 我的总结

这期视频最有价值的地方，是把迷信从“愚昧”还原成了“心理机制”。人会迷信，不只是因为知识少，也因为人会归纳、会焦虑、会害怕随机性、会想要控制感、会偷懒使用简单模型。

所以反迷信不是嘲笑别人，而是训练自己：

> 在随机中看见随机，在相关中谨慎寻找因果，在不确定中承受不确定。

真正的清醒不是从此没有任何仪式，而是知道哪些只是安慰，哪些正在吞掉你的判断力。
